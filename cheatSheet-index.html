HTML Reference for Tables

Table Tags

Tag Name Desciption
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing table data.
<tr> Table Row The table row container
<td> Table Data The data within the table row

Table Attributes

Attribute Name Desciption
<colspan> Column Span Defines how many colums your table should span.
<rowspan> Row Span Defines how many rows your table should span.
